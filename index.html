Celliverse — Hub

A cute, scientific exploration of cells & adaptation — Arctic Fox (animal) and Venus Flytrap (plant)

Start a new mission: explore organelles, play each organelle's mini-game, and watch the cell adapt for its environment. Each organism has 10 organelles with unique interactions.

Arctic Fox — Animal Cell

Adaptation focus: cold, fat storage, energy, insulation — complete organelle games to make the cell cold-resistant.
Launch Arctic Fox Mission
Enter codename on the mission page. Each organelle has a real mini-game and increases an adaptation meter. When the meter fills, see a mutation/evolution animation.

Venus Flytrap — Plant Cell

Adaptation focus: nutrient capture, photosynthesis efficiency, trap activation — complete organelle games to optimize the trap.
Launch Venus Flytrap Mission
Perfect for classroom demos — copy files into a GitHub Pages repo to publish. All code is client-side; works offline in a browser.
Tip: To host, push these three files to a GitHub repo and enable Pages from the repo settings. Happy exploring!
